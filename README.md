# SPL Feature Inference

A research prototype for LLM-assisted feature inference from product-ID-marked software variants.

## Purpose

This project analyzes outputs generated by an existing product integration tool.

The upstream tool produces:

- `model.xml`: asset-level ownership metadata
- `assets/`: product-ID-marked integrated asset files

This analyzer will extract variability evidence and generate inputs for feature candidate inference.

## Initial Milestone

The first milestone is to parse `input/model.xml` and generate:

- `output/asset_ownership_matrix.csv`

## Scope

This project does not implement product source integration.  
It assumes that product integration and product-ID marking have already been completed by an external tool.
